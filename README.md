# Library Interface Demo

This project implements a library system with two user roles:
- KidUser (age < 12, can borrow Kids books for 10 days)
- AdultUser (age > 12, can borrow Fiction books for 7 days)

## Files
- src/LibraryUser.java (interface)
- src/KidUser.java (KidUser class)
- src/AdultUser.java (AdultUser class)
- src/LibraryInterfaceDemo.java (main demo class with test cases)
